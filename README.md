# Bank Management Console App

A **Java-based console application** that simulates a real-world banking system using **object-oriented programming** principles.

## Key Features
✅ Demonstrates Object-Oriented Programming principles with real-world modeling
✅ Utilizes Java Collections (List, Map) for efficient data storage and management
✅ Implements custom exceptions to handle errors gracefully
✅ Enforces input validation for accurate and safe user operations
✅ Provides CRUD functionalities without relying on a database
✅ Simulates practical banking operations: account creation, deposits, withdrawals, transfers, and statements

## Functionalities
- Open Savings or Current Accounts  
- Deposit and Withdraw funds  
- Transfer money between accounts  
- View account statements  
- Search accounts by customer name  
- List all accounts  

## Technologies Used

- Java 17+  
- IntelliJ IDEA  
- Git & GitHub
